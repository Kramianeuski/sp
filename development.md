# System Power — brand site

## Идея проекта
Создать бренд‑портал System Power с SSR‑страницами, админ‑панелью для управления всеми текстами и контентом через базу данных, а также заделом под темы и модули по принципу Prestashop/OpenCart.

## Этапы разработки и изменения
1. Создан базовый каркас приложения на PHP с маршрутизацией публичной части и админ‑панели, подготовлены директории тем, модулей и хранилища.
2. Добавлены шаблоны темы, блоки страниц, базовая верстка и админ‑интерфейс для редактирования страниц, блоков, категорий, продуктов, файлов и переводов.
3. Подготовлена схема базы данных и стартовое наполнение контентом, навигацией, настройками сайта, продуктом и переводами.
4. Исправлен редирект языковых корней, чтобы исключить циклический переход /ru/ → /ru/.

## Seed-данные
- При каждой разработке добавляем новый файл seedN.sql с обновлениями.
- Не изменяем старые сиды — только создаём новый файл.
5. Добавлены новая схема БД и seed9.sql с пересозданием таблиц (кроме users), заполнены i18n-ключи, страницы/секции, SEO, категории и партнеры; добавлен импорт товаров из reference/product.txt с загрузкой изображений.
6. Переработаны маршруты, шаблоны и стили витрины под новую dark-тему с секциями, видео-hero, формой индивидуального производства и страницами качества/контактов/где купить.
7. Обновлена админка под новые таблицы (страницы, секции, категории, товары, партнеры, заявки, переводы) и добавлен импорт товаров.
8. Удален Banner.jpg из репозитория по требованию (используем poster как внешнюю/генерируемую ассетку при деплое).
